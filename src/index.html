Testing Backbone-Debugger

  1. Install Backbone Debugger
  2. If you can try do yourself
    1. Download this repo: git clone https://github.com/piecioshka/test-backbone-debugger.git
    2. Run bower install
    3. Open project in web browser Google Chrome
  3. Or without any installation
    1. Open http://piecioshka.pl/projects/test-backbone-debugger in web browser Google Chrome.
  4. Open DevTools -> Backbone and watch what will happen ;)
